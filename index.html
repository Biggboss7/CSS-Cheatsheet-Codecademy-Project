CSS CHEATSHEET

CSS Overview

  • CSS stands for Cascading Style Sheets
  • Used for styling HTML structure ex. decorating, positioning, controlling layout of elements
  • Preferred to be scripted in external files.

Basic Syntax

Selector {Property:Value;}

Example:

h1 {color:red;}

Typography

Attributes

  • font-family : typeface of an element
  • font-size : size of text displayed
  • font-weight : thin/thick of text
  • text-align : alignment of text

Color

Attributes

  • background-color: control color behind the text
  • color: color of the text
  • opacity: make element transparent
  • background-image: set an image into background

Color Representation

  • Color Name
  • RGB
  • HSL
  • Hexadecimal

Box Model

Attributes

  • margin : controls distance between element and outer elements
  • padding : controls distance between element and inside elements
  • border : The skin of the element
  • fixed : element outside HTML Flow, and fixed toone position
  • height : the height of the content
  • width : the width of the content
  • overflow : decide how content will be displayed if it overflow from its container

Display & Positioning

Positioning

Attributes

  • static : default, element in the HTML Flow
  • relative : element outside HTML Flow, but content height hold its previous position
  • absolute : element outside HTML Flow
  • fixed : element outside HTML Flow, and fixed to one position
  • sticky : element outside HTML Flow after reaches its position

Display

Attributes

  • block : element takes up entire block
  • inline : element only takes up needed space
  • inline-block : combination of inline and block
